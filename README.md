# Customer Churn Analysis Dashboard

## Project Overview
This project focuses on analysing customer churn behaviour using **Power BI** to identify the key factors influencing customer attrition and support better customer retention strategies.

The dashboard was built to provide business insights into customer behaviour, churn patterns, and service-related trends.

---

## Objective
The objective of this project was to:
- Understand why customers are leaving
- Identify patterns linked to churn
- Support data-driven retention strategies
- Present insights in a clear and interactive dashboard format

---

## Key Metrics
- **Total Customers:** 7,032
- **Churned Customers:** 1,869
- **Churn Rate:** 26.58%

---

## Analysis Performed
The analysis explored customer churn across several dimensions, including:

- **Contract Type**
- **Internet Service**
- **Payment Method**
- **Customer Tenure**

The goal was to compare churned and retained customers to understand which customer groups showed higher churn behaviour.

---

## Key Insights
- Customers on **month-to-month contracts** showed a higher churn tendency compared to customers on long-term contracts.
- Customers with **shorter tenure** were more likely to churn earlier in the customer lifecycle.
- Churn patterns varied across **internet service categories** and **payment methods**.
- These findings can help businesses improve **customer retention strategies** and reduce customer loss.
